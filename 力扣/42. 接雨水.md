给定 `n` 个非负整数表示每个宽度为 `1` 的柱子的高度图，计算按此排列的柱子，下雨之后能接多少雨水。
![[Pasted image 20260207150259.png]]

```
/**

* @param {number[]} height

* @return {number}

*/

var trap = function(height) {

let left = 0

let right = height.length - 1

let maxL = height[left]

let maxRight = height[right]

let res = 0

while(left < right) {

if (maxL < maxRight) {

left = left + 1

maxL = Math.max(maxL, height[left])

res = res + (maxL - height[left])

} else {

right = right - 1

maxRight = Math.max(maxRight, height[right])

res = res + (maxRight - height[right])

}

}

return res

};
```

重点：
- 接的雨水是 max(左指针高度， 右指针高度) - 当前指针的高低(只计算高度就行，因为宽度肯定是1)，记得得先挪当前指针也就是left = left + 1或者right = right - 1
