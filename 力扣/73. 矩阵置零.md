给定一个 `_m_ x _n_` 的矩阵，如果一个元素为 **0** ，则将其所在行和列的所有元素都设为 **0** 。请使用 **[原地](http://baike.baidu.com/item/%E5%8E%9F%E5%9C%B0%E7%AE%97%E6%B3%95)** 算法

```
**示例 1：**

![](https://assets.leetcode.com/uploads/2020/08/17/mat1.jpg)

**输入：**matrix = [[1,1,1],[1,0,1],[1,1,1]]
**输出：**[[1,0,1],[0,0,0],[1,0,1]]

**示例 2：**

![](https://assets.leetcode.com/uploads/2020/08/17/mat2.jpg)

**输入：**matrix = [[0,1,2,0],[3,4,5,2],[1,3,1,5]]
**输出：**[[0,0,0,0],[0,4,5,0],[0,3,1,0]]
```

```js
/**

* @param {number[][]} matrix

* @return {void} Do not return anything, modify matrix in-place instead.

*/

var setZeroes = function (matrix) {
  let firstRowZero = false // 记录第一行是否含0

  let firstColZero = false // 记录第一列是否含0

  const rowLen = matrix.length // 第一行的长度

  const colLen = matrix[0].length // 第一列的长度

  for (let i = 0; i < colLen; i++) {
    // 第一行是否有0

    if (matrix[0][i] === 0) {
      firstRowZero = true
    }
  }

  for (let j = 0; j < rowLen; j++) {
    // 第一列是否有0

    if (matrix[j][0] === 0) {
      firstColZero = true
    }
  }

  // 用第一行和第一列做标记

  for (let i = 1; i < rowLen; i++) {
    for (let j = 1; j < colLen; j++) {
      if (matrix[i][j] === 0) {
        matrix[i][0] = 0

        matrix[0][j] = 0
      }
    }
  }

  // 根据标记清零内部区域

  for (let i = 1; i < rowLen; i++) {
    for (let j = 1; j < colLen; j++) {
      if (matrix[i][0] === 0 || matrix[0][j] === 0) {
        matrix[i][j] = 0
      }
    }
  }

  // 处理第一行

  for (let i = 0; i < colLen; i++) {
    if (firstRowZero) {
      matrix[0][i] = 0
    }
  }

  // 处理第一列

  for (let i = 0; i < rowLen; i++) {
    if (firstColZero) {
      matrix[i][0] = 0
    }
  }

  return matrix
}

```
1. 不再新建 `row[]` 和 `col[]` 数组
2. **直接复用 `matrix` 的第 0 行和第 0 列**，来表示某行、某列是否需要被置零：
    - 如果在扫描时发现 `matrix[i][j] === 0`（`i>0, j>0`）
        - 把 `matrix[i][0]` 设为 0（表示第 i 行要变 0）
        - 把 `matrix[0][j]` 设为 0（表示第 j 列要变 0）
3. 问题：那**原本第 0 行、第 0 列自己是不是该变 0**，怎么记？
    - 再单独用两个布尔变量记：
        - `firstRowHasZero`：第一行原本是否有 0
        - `firstColHasZero`：第一列原本是否有 0

这样就实现了：

- “标记数组”存在矩阵里（第一行、第一列）
- 真正额外用的只有两个布尔变量 → **O(1) 空间**
